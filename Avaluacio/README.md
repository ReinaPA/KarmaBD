# 📋 Plantilla d'Avaluació

Aquesta <a href="https://github.com/ReinaPA/KarmaBD/blob/main/Avaluacio/Plantilla%20avaluaci%C3%B3%20x%20grup%20BBDD.xlsx" target="_blank">plantilla</a> està dissenyada per facilitar l'avaluació individual de l'alumnat.

## 🧑‍🏫 Instruccions d'ús

1. **Creació de fitxers**  
   Cal crear una còpia de la plantilla per a **cada alumne**.

2. **Introducció de notes**  
   Les notes de les evidències s'han d'introduir a la fulla anomenada **"Evidències"**.

3. **Afegir nous instruments d'avaluació**  
   - Si s'utilitzen més instruments dels inicialment previstos, cal:
     - Afegir-los a les fulles dels **Resultats d'Aprenentatge (RA)** corresponents.
     - Assignar-los la **ponderació** adequada segons la seva importància.

---

📌 *Assegura't de revisar que totes les ponderacions estiguen ben ajustades per garantir una avaluació justa i coherent.*
